What is the internet?
The internet is a network of interconnected computers. A server is a computer that stores information that the user wants to access. A client is a computer the user interacts with directly. Together all connected servers and clients in the world make up the internet HTTP stands for HyperText Transfer Protocol. HTTP is what allows clients and servers to communicate with each other
What is HTML?
HTML stands for HyperText MakeUp Language and it's the primary language of the internet. An HTML file consist of a series of elements defined by tags. A tag is a set of characters enclosed between a "less than" sign and a "greater than" sign . Normally there is tag and a clossing tag, although there are some exeptions. The content from the beginning of the opening tag to the end of the closing tag comprises one element.(in those cases where there is no closing tag, the one tag comprises the entire)
